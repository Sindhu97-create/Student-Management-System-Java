# Student Management System - Java

## Project Overview

The Student Management System is a console-based Java application developed to manage student records efficiently. The system allows users to add, view, search, update, and delete student information using Core Java concepts.

## Objectives

- Manage student records efficiently.
- Perform CRUD operations on student data.
- Apply Object-Oriented Programming concepts.
- Improve understanding of Java fundamentals.

## Technologies Used

- Java
- OOP Concepts
- ArrayList
- Eclipse / IntelliJ IDEA

## Features

- Add Student
- View Students
- Search Student
- Update Student Details
- Delete Student
- Exit Application

## Outcome

This project helped strengthen Java programming skills, object-oriented design principles, and data management techniques.
